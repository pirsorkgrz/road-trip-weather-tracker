# Road Trip Weather Tracker

## Overview
This repository tracks weather conditions for a planned cross‑country road trip from Los Angeles, CA to New York, NY in July 2025. It gathers active weather alerts and forecasts for the departure and destination states, providing a centralized view for trip planning.

## Monitored States
- California (departure)
- New York (arrival)

## Purpose
- Monitor heat advisories and other alerts in California before departure.
- Monitor flood, wind, and other alerts in New York before arrival.
- Keep the data up‑to‑date for the trip timeline.
